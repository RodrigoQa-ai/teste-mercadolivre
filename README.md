# Projeto de Testes Manuais — Mercado Livre

## Objetivo

Realizar testes manuais exploratórios no site Mercado Livre com o objetivo de validar funcionalidades principais de um e-commerce, identificar falhas de comportamento e documentar possíveis defeitos.

## Ambiente de Teste

* Navegador: Google Chrome
* Sistema Operacional: Windows 10, Android
* Dispositivo: Desktop e Mobile
* Data dos testes: 12-02-2026
## Escopo

As seguintes funcionalidades foram avaliadas:

* Busca de produtos
* Aplicação de filtros
* Ordenação de resultados
* Visualização de página do produto
* Adição e remoção do carrinho
* Navegação entre páginas

## Cenários de Teste

1. Buscar produto válido
2. Buscar produto inexistente
3. Aplicar filtro de preço
4. Ordenar por menor preço
5. Abrir página de produto
6. Adicionar produto ao carrinho
7. Remover produto do carrinho
8. Atualizar página do carrinho
9. Navegar usando botão "voltar" do navegador
